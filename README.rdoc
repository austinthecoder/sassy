= Sassy

Generate SCSS using ruby.

== Example

  Sassy::SCSS.build do
    body do
      background '#000'
      color '#fff'
    end

    r '#header' do
      background 'green'
      d 'font-size', '13px'

      a do
        color :red;
      end

      r '#logo' do
        background :blue
      end
    end
  end
